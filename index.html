Survol du langage HTML

C'est quoi le HTML ?

HyperText Markup Language — c'est un langage de description, pas de programmation.

HTML vs CSS HTML décrit la structure du document. CSS se charge de la présentation.

Les balises

Une balise est délimitée par < et >. Elle a une ouvrante et une fermante.

<p> Mon chat est fâché </p> <br />

Structure de base

<html> <head> <title>Titre</title> </head> <body>Contenu</body> </html>

Survol du langage CSS

Rôle du CSS

CSS gère la présentation. Un même HTML peut avoir plusieurs apparences.

Où écrire le CSS ?

MéthodeExemple
Balise <style><style type="text/css">…</style>
Fichier externe<link rel="stylesheet" href="style.css"/>
Inline<p style="color:red;">

Syntaxe d'une règle CSS

h1 { color: blue; text-align: center; }

Les types de sélecteurs

p { margin: 0; } /* balise */ .sommaire { color: red; } /* classe */ #monId { background: yellow; } /* id */

Pseudo-classes

a:hover { text-decoration: none; } a.lien:hover { color: red; }

Regroupement

.texte, p, h1 { margin-left: 0; } div p { color: gray; }

Démo — Style en direct

Le CSS change l'apparence sans toucher au HTML.
