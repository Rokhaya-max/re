Survol du langage HTML

C'est quoi le HTML ?

HyperText Markup Language — un langage de description, pas de programmation. Il crée des pages web lisibles dans un navigateur.

HTML vs CSSHTML décrit la structure. CSS gère la présentation. Le W3C recommande de toujours séparer les deux.

Les balises

Une balise est délimitée par < et >. Elle a une ouvrante et une fermante. Les balises orphelines se ferment elles-mêmes.

<p> Mon chat est fâché </p> — balise normale <br /> — balise orpheline

Structure de base d'un document

<!DOCTYPE html> <html> <head> <title>Titre de la page</title> </head> <body> Ce qui s'affiche dans le navigateur </body> </html>

Survol du langage CSS

Rôle du CSS

CSS (Cascading Style Sheets) gère la présentation. Un même fichier HTML peut avoir plusieurs apparences avec différentes feuilles de style.

Où écrire le CSS ?

MéthodeExemple
Balise <style><style type="text/css">…</style>
Fichier externe<link rel="stylesheet" href="style.css"/>
Attribut inline<p style="color:red;">

Syntaxe d'une règle CSS

Une règle CSS = un sélecteur + une déclaration entre accolades.

h1 { color: blue; text-align: center; }

Les types de sélecteurs

— sélecteur de balise — p { margin-top: 0; margin-bottom: 0; } — sélecteur de classe (préfixé d'un point) — .sommaire { color: red; } <a class="sommaire">lien rouge</a> — sélecteur d'identifiant (préfixé d'un #) — #monId { background-color: yellow; } <div id="monId">contenu</div>

Pseudo-classes

— survol d'un lien — a:hover { text-decoration: none; } — classe précise au survol — a.lien:hover { text-decoration: none; color: red; }

Regroupement de sélecteurs

— même style pour plusieurs balises — .texte, p, h1, h2 { margin-left: 0; } — sélection hiérarchique : p dans un div — div p { color: gray; } — propriété raccourcie — p { font: bold 14px Arial; }
Propriétés raccourcies utilesfont · border · background — elles regroupent plusieurs valeurs en une seule déclaration.

Le langage JavaScript

C'est quoi JavaScript ?

JS est un langage de script incorporé dans le HTML. Il rend le contenu interactif : validation de formulaires, animations, boites de dialogue, etc.

Où placer le script ?

— dans le body — <script type="text/javascript"> // code ici </script> — fichier externe — <script src="mon_script.js"></script>
