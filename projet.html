Survol du langage HTML

C'est quoi le HTML ?

HyperText Markup Language — c'est un langage de description, pas de programmation. Il permet de créer des pages web lisibles dans un navigateur.

HTML vs CSS HTML décrit la structure du document. CSS se charge de la présentation. Le W3C recommande de toujours séparer les deux.

Les balises

Une balise est délimitée par < et >. Elle a une ouvrante et une fermante. Les balises orphelines se ferment elles-mêmes.

<p> Mon chat est fâché </p> -- balise normale <br /> -- balise orpheline (pas de contenu)

Structure de base d'un document XHTML

Tout document HTML/XHTML suit cette structure :

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"> <html> <head> <title>Titre de la page</title> </head> <body> Ce qui s'affiche dans le navigateur </body> </html>

Les liens

Un lien hypertexte envoie le visiteur vers une autre page ou une autre zone de la page.

-- lien vers une autre page -- <a href="destination.html">Texte du lien</a> -- lien externe -- <a href="http://www.google.sn" target="_blank">Google</a> -- lien email -- <a href="mailto:webmaster@monsite.sn">Ecrivez-moi</a> -- ancre (navigation dans la page) -- <a href="#motClef">Aller au chapitre</a> <h2 id="motClef">Chapitre</h2>

Survol du langage CSS

Rôle du CSS

CSS (Cascading Style Sheets) se charge de la présentation. Un même fichier HTML peut avoir plusieurs apparences avec différentes feuilles de style.

Où écrire le CSS ?

MéthodeExemple
Balise <style> dans <head><style type="text/css">…</style>
Fichier externe via <link><link rel="stylesheet" href="style.css"/>
Attribut style inline<p style="color:red;">

Syntaxe d'une règle CSS

Une règle CSS = un sélecteur + une déclaration entre accolades.

h1 { color: blue; text-align: center; }

Les types de sélecteurs

-- sélecteur de balise -- p { margin-top: 0; margin-bottom: 0; } -- sélecteur de classe (préfixé d'un point) -- .sommaire { color: red; } <a class="sommaire">lien rouge</a> -- sélecteur d'identifiant (préfixé d'un #) -- #monId { background-color: yellow; } <div id="monId">contenu</div>

Pseudo-classes et pseudo-éléments

-- survol d'un lien -- a:hover { text-decoration: none; } -- lien d'une classe précise au survol -- a.lien:hover { text-decoration: none; color: red; }

Regroupement de sélecteurs

-- appliquer le même style à plusieurs balises -- .texte, p, h1, h2 { margin-left: 0; } -- sélection hiérarchique : p dans un div -- div p { color: gray; } -- propriété raccourcie : font -- p { font: bold 14px Arial; }
Propriétés raccourcies utiles font · border · background — elles regroupent plusieurs valeurs en une seule déclaration.

Le langage JavaScript

C'est quoi JavaScript ?

JS est un langage de script incorporé dans le HTML. Il rend le contenu interactif : validation de formulaires, animations, boites de dialogue, etc.

Où placer le script ?

-- dans le body (sans fonctions) -- <body> <script type="text/javascript"> // code ici </script> </body> -- fichier externe -- <script src="mon_script.js"></script>

Variables — déclaration et initialisation

var date; // déclaration sans affectation var compteur = 0; // déclaration avec affectation nom = 'Diop'; // déclaration implicite var prem = 5, second = 18; // plusieurs variables var maChaine = ""; // chaine vide var isOk = true; // booléen
